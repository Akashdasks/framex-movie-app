# 🎬 FrameX – Movie Discovery App

FrameX is a modern movie discovery web application built using React and the TMDB API.  
It allows users to explore trending movies, browse by categories, and manage a personal watchlist.

---

## 🚀 Features

- 🔥 Trending Movies
- 🎭 Browse Movies by Genre
- 🎬 Detailed Movie View
- ⭐ Add / Remove from Watchlist
- 🌙 Dark Theme UI
- ⚡ Fast and Responsive Design

---

## 🛠 Tech Stack

- React
- Vite
- TMDB API
- CSS (Custom Styling)
- Git & GitHub
